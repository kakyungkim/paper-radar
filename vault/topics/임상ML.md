# 🗂 임상ML — 주제 지도(MOC)

## 핵심 흐름
(2026-W24 기준) 액체생검 기반 치료 내성 조기 감시가 통계 모델링 혁신으로 정밀해지고 있다. 검출 한계 아래 데이터를 결측이 아닌 구간 중도절단(censored) 관측으로 처리하는 베이지안 변화점 모델이 ctDNA 직렬 검사에서 내성 서브클론 출현을 임상보다 먼저 포착한다. 병리 AI 쪽에서는 범용 파운데이션 모델이 조직 형태의 복잡성을 처리하기에 부적합하다는 비판적 관점(Nature Biomedical Engineering)이 제기되며 병리 전용 아키텍처 논의가 본격화됐다.
