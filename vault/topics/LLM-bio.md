# 🗂 LLM-bio — 주제 지도(MOC)

## 핵심 흐름
(2026-W24 기준) 생물학 특화 파운데이션 모델이 RNA 변형(m6A)·분자 특성·가상 세포·디지털 트윈 등 여러 하위 도메인에서 동시에 등장하며 "생물학 AI의 모달리티 분화" 국면이 뚜렷해졌다. GLACIER와 OCOO-T·m6A-FORM 모두 단일 트랜스포머 아키텍처 위에 도메인 특화 사전학습 전략을 얹는 패턴을 따른다. 코드 미공개 비율이 핵심 5편 중 3편으로 높아 재현성 우려가 남는다.
