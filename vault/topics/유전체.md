# 🗂 유전체 — 주제 지도(MOC)

## 핵심 흐름
(2026-W25 기준) 단일세포(single-cell) 분석이 그래프 구조 학습과 트랜스포머를 결합하는 방향으로 진화하고 있다. scGTN은 샴(Siamese) 네트워크로 세포 간 그래프 유사도를 명시적으로 학습해 scRNA-seq 클러스터링의 희소성·잡음 문제에 대응한다. 멀티오믹스 암 분석에서는 전체 슬라이드 이미지(WSI)와 전사체를 결합한 파운데이션 모델 표현이 실제 상업 코호트에서 분포 이동 취약성을 드러내며, 유전체 데이터 기반 병리 AI의 배포 한계를 구체화한다. W24의 RNA 후성유전체·가상 세포 트렌드에서 이어, W25에는 세포 클러스터링 아키텍처 혁신이 주 흐름이다.

- **scGTN(Wu 외)** — 샴 그래프 트랜스포머로 세포 간 구조 정보 통합. scRNA-seq 클러스터링. 핵심 수치 초록 기준 미기재. 코드 미공개(논문 내 URL 명시 예고). 출처: [[digest/2026-W25]] | arXiv:2606.18672
- **멀티모달 암 FM 벤치마킹(Hu 외)** — WSI+전사체 프로파일 통합. 유방암(IH-BC)·비소세포폐암(IH-NSCLC) 실제 상업 코호트. 분포 이동 하 일반화 취약 확인. 출처: [[digest/2026-W25]] | arXiv:2606.17115
