## 1. m6A-FORM: N6-메틸아데노신 예측 파운데이션 모델

**논문**: "A Foundation Model for Decoding N6-methyladenosine Biology"  
**링크**: https://arxiv.org/abs/2606.12219  
**상태**: arXiv preprint (미동료심사) · 2026-06-10  
**코드/데이터**: 미제공 — arXiv 페이지 및 초록에 GitHub 또는 데이터 저장소 링크 없음

---

### 무엇이 새로운가

**저자가 주장하는 새로운 점**:
- MeRIP-seq 피크를 "메틸화 강화 사전(methylation-enriched prior)"으로 삼아 약 2,200만 개 피크 파생 서열(143개 인간 연구)로 사전학습한 트랜스포머.
- 단일 뉴클레오타이드 수준 m6A 사이트 예측 — 기존 피크(peak) 수준 예측에서 한 단계 더 고해상도.
- 19종 m6A 관련 조절인자(YTHDF1/2/3 등 리더·작성자·지우개) 결합 부위를 단일 모델에서 동시 예측.

**실제로 새로운 점 구분**:
- 외부 맥락: m6A 예측 모델(MetTL2D 등)은 이전에도 있었으나, MeRIP-seq 자체를 사전학습 코퍼스로 대규모 활용한 파운데이션 모델 접근은 차별화 포인트다.
- 해석: 22백만 시퀀스 사전학습 후 고신뢰도 단일 뉴클레오타이드 주석(m6A-Atlas v2.0 + GLORI)으로 파인튜닝하는 2단계 구조는 NLP 사전학습-파인튜닝 패러다임을 RNA 후성유전체에 이식한 것이며, 이 자체는 신규성이 있다.
- 검토필요: "기존 최고 대비 PR-AUC +0.14 이상"이라는 수치는 초록 기준이며, 어떤 베이스라인 모델과 비교했는지 논문 본문 없이 확인 불가.

---

### 핵심 가정

1. **MeRIP-seq 피크 = 신뢰 가능한 m6A 신호**: MeRIP-seq는 항체 기반으로 위양성이 약 30~50% 수준이라는 외부 맥락이 있다. 노이즈가 심한 피크를 대규모로 사전학습에 쓰면 모델이 노이즈 패턴을 학습할 위험이 있다.
2. **143개 연구의 프로토콜 동질성**: 연구마다 항체·라이브러리 제조 프로토콜이 다르다. 이를 하나의 코퍼스로 합산하는 것은 배치 효과(batch effect) 혼입을 전제로 감내한다.
3. **단일 뉴클레오타이드 해상도 파인튜닝 데이터의 대표성**: m6A-Atlas v2.0 + GLORI는 고신뢰도 데이터이나 특정 세포주·조직 편향이 있을 수 있다(외부 맥락). 24개 조직 일반화가 이 편향을 극복하는지는 미제공.

**가정이 깨질 때**: MeRIP-seq 노이즈가 높은 세포 유형에서 사전학습 편향이 단일 뉴클레오타이드 예측 정확도를 낮추고, PR-AUC 이점이 사라질 수 있다.

---

### 재현 가능성

| 항목 | 상태 |
|------|------|
| 코드 | 미제공(초록 기준) |
| 모델 가중치 | 미제공 |
| 학습 데이터 | 공개 MeRIP-seq(GEO 등) 143개 연구에서 수집 — 재수집은 가능하나 전처리 파이프라인 미공개 |
| 파인튜닝 데이터 | m6A-Atlas v2.0 + GLORI — 공개 데이터셋이나 정확한 분할 미기재 |
| 베이스라인 공정성 | 미제공 — 어떤 구현 버전의 어떤 베이스라인을 썼는지 초록에 없음 |
| 데이터 누수 신호 | 질문: m6A-Atlas와 143개 MeRIP-seq 연구 간 중복 샘플 여부 미확인. 누수 가능성 배제 불가 |

---

### 검증 수준

- in silico 검증: PR-AUC, ROC-AUC 보고 (초록 기준).
- 24개 조직에서 19,631개 보존 사이트 식별 — 내부 응용 분석, 독립 외부 데이터셋 검증 여부 미제공.
- 19종 조절인자 결합 부위 예측 — 실험적(wet-lab) 검증 여부 미제공.
- ablation: 미제공(초록 기준).

해석: PR-AUC 0.635는 절댓값으로 낮아 보이지만, m6A 사이트 예측에서 극도로 불균형한 양성:음성 비율(전체 DRACH 모티프 중 실제 m6A는 극소수) 탓에 이 정도면 상당한 수준일 수 있다. 그러나 이 해석 역시 본문 없이 초록 기준임을 명시한다.

---

### 강점 / 한계

**강점**
- 대규모(22M 시퀀스) RNA 특화 사전학습으로 in-context generalization 기대.
- 다중 조절인자 예측 통합 — 파이프라인 단순화 가능.
- m6A-Atlas v2.0 + GLORI 파인튜닝 = 현재 가장 고신뢰도 레이블 활용.

**한계**
- 코드·가중치 미공개로 재현 불가(현시점).
- 베이스라인 선택 및 비교 조건 불투명(초록 기준).
- MeRIP-seq 노이즈 문제를 사전학습 단계에서 어떻게 처리했는지 미제공.
- 단백질-m6A 리더 결합 예측의 실험 검증 없음(초록 기준).

---

### 내 연구 적용 메모 (전략 노트)

- 전략 노트: RNA 후성유전체 파이프라인에서 m6A 사이트 예측이 필요한 경우, 코드 공개 시 즉시 벤치마크 후보. 현재는 코드 없이 적용 불가.
- 외부 맥락: m6A는 mRNA 안정성·번역 효율·pre-mRNA 스플라이싱에 영향을 주므로 RNA 치료제(ASO, mRNA 의약품) 설계 시 고려 가능한 변형이다.
- 질문: 비인간 종(마우스 등) 또는 ncRNA로 전이 학습 가능한지 초록에 언급 없음. 공개되면 확인 필요.

## 2. OCOO-T: 흐름 매칭 기반 가상 세포 교란 반응 예측

**논문**: "A Simple and Scalable Virtual Cell Model for Transcriptional Perturbation Response Prediction"  
**링크**: https://arxiv.org/abs/2606.12838  
**상태**: arXiv preprint (미동료심사) · 2026-06-11  
**코드/데이터**: 미제공 — arXiv 페이지 및 초록에 저장소 링크 없음

---

### 무엇이 새로운가

**저자가 주장하는 새로운 점**:
- 흐름 매칭(flow matching) 기반 생성 모델로 교란 반응 예측 — 기존 VAE/확산 모델 대신.
- 보조 인코더 없이 "바닐라 트랜스포머"가 연속 유전자 발현 프로필에 직접 작동(auxiliary encoder 불필요).
- 적응형 레이어 정규화(adaptive layer normalization) + 컨텍스트 토큰으로 교란 임베딩·용량·세포 유형 특이성 통합 — 별도 세포 유형 인코더 없이 단일 모델에서 다중 교란 유형(유전적·화학적·사이토카인) 처리.

**실제로 새로운 점 구분**:
- 외부 맥락: Geneformer, scGPT, GEARS 등 기존 가상 세포 모델들은 복잡한 계층적 아키텍처를 가진다. "단순하다"는 주장은 방법론적 절제(Occam's razor)의 관점에서 실제 강점이 될 수 있다.
- 해석: 흐름 매칭을 교란 반응 예측에 적용한 사례는 scFM(2024) 이후 증가하는 추세이나, 이 논문은 세 가지 교란 유형(유전적·화학적·사이토카인)을 단일 아키텍처로 처리한다는 점이 차별점으로 읽힌다. 단, 초록 기준 확인임을 명시.
- 검토필요: "OCOO-T"라는 이름의 의미(약어 풀이) 초록에 미설명.

---

### 핵심 가정

1. **교란 반응이 연속 흐름으로 모델링 가능**: 세포 상태 전이를 연속 변환(flow)으로 보는 것은 세포 생물학적 이산성(세포 사멸, 분화 분기)을 단순화한다.
2. **컨텍스트 토큰이 세포 유형 특이성을 충분히 포착**: 세포 유형 ID를 토큰으로 표현하면 희귀 세포 유형이나 환자 특이적 변이는 과소대표될 수 있다.
3. **Tahoe100M · Replogle · PBMC 벤치마크가 실제 임상 교란을 대표**: 이 벤치마크들은 체계적으로 설계된 실험 데이터로 실제 환자 반응 이질성을 반영하지 않을 수 있다(외부 맥락).

**가정이 깨질 때**: 희귀 세포 유형이나 novel 화학 공간에서 흐름 매칭의 일반화 성능 저하 가능. 세포 사멸 유도 교란에서 연속 흐름 가정 자체가 성립하지 않을 수 있다.

---

### 재현 가능성

| 항목 | 상태 |
|------|------|
| 코드 | 미제공 |
| 모델 가중치 | 미제공 |
| 학습 데이터 | Tahoe100M(공개), Replogle(공개), PBMC(공개) — 데이터는 재수집 가능 |
| 벤치마크 수치 | 미제공 — 초록에 구체적 수치 없음, "경쟁력 있는 성능"만 기재 |
| 베이스라인 공정성 | 질문: 어떤 버전의 Geneformer/scGPT와 비교했는지 초록에 없음 |
| 데이터 누수 | 질문: Tahoe100M 사전학습 시 테스트 교란이 학습 셋에 포함됐는지 불명 |

---

### 검증 수준

- in silico 3개 벤치마크 검증: Tahoe100M, Replogle, PBMC.
- 구체적 수치 미제공(초록 기준) — "경쟁력 있는 성능"이라는 정성 표현만.
- 외부 독립 데이터셋 검증: 미제공.
- ablation: 미제공(초록 기준) — adaptive layer norm vs. 표준 layer norm, context token 제거 실험 여부 불명.

해석: 수치가 초록에 없다는 것은 결과가 드라마틱하지 않을 가능성이 있거나, 혹은 preprint 특성상 저자가 의도적으로 논문 본문으로 유도하는 것일 수 있다. 어느 쪽이든 현재 초록 기반으로는 정량 판단 불가.

---

### 강점 / 한계

**강점**
- 세 가지 교란 유형을 단일 모델로 처리 — 파이프라인 통합 편의성.
- 공개 벤치마크 활용으로 향후 비교 가능성 확보.
- "단순함"을 명시적 설계 원칙으로 삼아 엔지니어링 복잡도 절감 주장.

**한계**
- 코드·가중치 미공개.
- 초록에 구체적 성능 수치 없음 — 현 단계에서 정량 평가 불가.
- 실제 약물 스크리닝 시나리오(novel compound, out-of-distribution) 일반화 미검증.
- wet-lab 실험 검증 없음(in silico only, 초록 기준).

---

### 내 연구 적용 메모 (전략 노트)

- 전략 노트: 유전자 편집(CRISPR) 또는 약물 처리 반응 예측 파이프라인이 필요한 프로젝트에서, 코드 공개 시 GEARS/scGPT 대비 단순성 이점이 있는지 직접 비교할 만하다.
- 외부 맥락: Replogle 등 대규모 교란 데이터셋의 품질이 가상 세포 모델 전반에 병목이므로, 데이터 품질 논의가 방법 논의만큼 중요하다.
- 질문: 임상 약물(FDA 승인 화합물 등)에 대한 반응 예측 정확도가 체계적으로 보고되는지 초록에 없음.

## 3. Span Detector: 검출 한계 이하 ctDNA 변화점 감지

**논문**: "Seeing Below the Limit of Detection: A Censored-Poisson Bayesian Latent-Growth Change-Point Detector for Serial ctDNA in HR+/HER2- Metastatic Breast Cancer"  
**링크**: https://arxiv.org/abs/2606.11876  
**상태**: arXiv preprint (미동료심사) · 2026-06-10  
**코드/데이터**: 공개 — https://github.com/span-ai-labs/span-detector

---

### 무엇이 새로운가

**저자가 주장하는 새로운 점**:
- 비검출(undetected) ctDNA 측정값을 "결측" 또는 "0"으로 버리지 않고 **좌측 구간 중도절단(left-censored)** 관측으로 처리.
- 포아송 이진 검출 과정 + 베이지안 잠재 성장 모델 + 변화점 탐지를 하나의 계층 모델에 통합.
- 경보 제어(competitive risk alarm)를 보정해 허위 경보율(FAR) 조건부로 조기 검출률 향상.

**실제로 새로운 점 구분**:
- 외부 맥락: 구간 중도절단 처리는 생존분석에서 표준이지만, ctDNA 직렬 측정값에 Censored-Poisson + 잠재 성장 + 변화점을 결합한 통합 모델은 문헌에서 드물다.
- 해석: 이 논문의 진짜 기여는 **데이터 재정의** — "비검출 = 정보"라는 관점 전환이 핵심이고, 통계 모델은 그 관점을 구현하는 수단이다. 아키텍처 자체보다 문제 프레이밍이 더 새롭다.
- 검토필요: "HR+/HER2- 전이성 유방암"이라는 구체적 적응증에 모델이 특화됐는지, 아니면 일반 ctDNA 직렬 데이터에 적용 가능한지 초록 기준으로 불명확.

---

### 핵심 가정

1. **ctDNA 검출이 포아송 과정을 따른다**: 실제로 ctDNA 단편화와 포획(capture) 효율이 포아송 모델 가정에서 벗어나는 경우(증폭 편향 등)가 있다(외부 맥락).
2. **잠재 성장이 선형·연속적**: 내성 서브클론의 성장이 지수적(exponential)이거나 단속적(punctuated)일 경우, 잠재 선형 성장 가정이 적합하지 않을 수 있다.
3. **변화점이 단일**: 치료 과정에서 여러 내성 클론이 순차·병렬로 출현하는 다중 변화점 시나리오에서의 성능 미확인.
4. **합성 시나리오가 실제 환자 데이터를 대표**: 조기 검출률 25% vs 11%는 합성 데이터 기준(초록 명시). 실제 코호트 검증은 GBSG-2·PBC2이지만, 이 코호트는 원래 ctDNA 직렬 검사 연구가 아니라 생존분석 벤치마크다.

**가장 취약한 가정**: 합성 데이터 기반 성능 주장. GBSG-2(n=686), PBC2(n=312)의 C-index 0.67 vs 0.68은 개선이 없거나 미미한 수준이다(초록 수치).

해석: 조기 검출 이점은 합성 데이터에서만 명확히 나타나고, 실제 코호트에서는 C-index가 기저치와 거의 같다. "2배 향상"이라는 표현은 합성 데이터에 한정된 주장으로 수위 조절이 필요하다.

## 4. GLACIER: 멀티모달 학생-교사 분자 특성 예측

**논문**: "A Multimodal Student-Teacher Foundation Model for Molecular Property Prediction"  
**링크**: https://arxiv.org/abs/2606.11382  
**상태**: arXiv preprint (미동료심사) · 2026-06-09  
**코드/데이터**: **공개** — https://github.com/eemokey/glacier

---

### 무엇이 새로운가

**저자가 주장하는 새로운 점**:
- 분자 그래프(GNN) + SMILES(트랜스포머) + 물리화학 기술자(MLP)를 세 학생 인코더로 동시 학습 후 **지식 증류(knowledge distillation)**로 단일 모델에 통합.
- Finsler 기하학 기반 멀티모달 융합 모듈 — 비유클리드 거리 측도로 모달리티 간 정보 통합.
- 교사 모델로 MiniMol, MolFormer 등 대형 사전학습 모델을 사용해 소규모 학습 데이터에서 지식 전달.
- 10만 개 약물 유사 분자 사전학습 — 파라미터 대비 효율 강조.

**실제로 새로운 점 구분**:
- 외부 맥락: 멀티모달 분자 표현(그래프+SMILES)은 MolBERT, UniMol 등에서 이미 시도됐다. GLACIER의 차별점은 물리화학 기술자를 세 번째 모달리티로 추가하고 Finsler 기하학 융합을 도입한 것이다.
- 해석: "학생-교사 지식 증류"는 NLP에서 확립된 기법이다. 분자에 적용한 사례는 있으나, 세 가지 이질적 모달리티에 동시 적용한 구현은 차별화 요소다.
- 검토필요: Finsler 기하학 융합의 실질적 이점이 ablation으로 검증됐는지 초록에 미기재.

---

### 핵심 가정

1. **세 모달리티가 상호 보완적**: 그래프·SMILES·물리화학 기술자가 서로 다른 정보를 담는다고 가정. 그러나 세 표현은 동일 분자를 다르게 인코딩할 뿐 정보 내용의 독립성이 보장되지 않는다.
2. **10만 개 사전학습 분자가 목표 화학 공간을 커버**: ADME 관련 화학 공간은 매우 넓으며, 10만 개로 전체를 대표하기는 어렵다(외부 맥락). CHEMBL에는 200만 개 이상 등록.
3. **교사 모델(MiniMol, MolFormer)의 지식이 전이 가능**: 교사가 특정 화학 공간에서 학습됐다면, 분포가 다른 ADME 화합물 공간에서 지식 전달이 효과적이지 않을 수 있다.
4. **Finsler 기하학이 분자 표현 공간의 구조를 포착**: 이론적 근거는 있으나 분자 ML에서 실증적으로 검증된 사례가 드물다(외부 맥락).

---

### 재현 가능성

| 항목 | 상태 |
|------|------|
| 코드 | **공개** — https://github.com/eemokey/glacier |
| 사전학습 데이터 | "10만 약물 유사 분자" — 정확한 소스 미기재(추정: ZINC/ChEMBL 하위셋) |
| 벤치마크 | Biogen, ExpansionRX, ChEMBL-MT — 일부 내부 벤치마크 포함 가능성 |

| 데이터 누수 | 질문: Biogen/ExpansionRX 벤치마크가 내부 데이터인 경우 독립 검증 불가 |

---

### 검증 수준

- 3개 벤치마크 검증: Biogen(+7.6%), ExpansionRX(+9.9%), ChEMBL-MT(+9.5%).
- 기저치(KERMT) 대비 상대적 개선 — 절대 수치 미기재(초록 기준).
- 외부 공개 독립 벤치마크(MoleculeNet 등) 검증 여부 미제공.
- ablation: 미제공(초록 기준) — Finsler 융합, 지식 증류, 각 모달리티 제거 실험 여부 불명.

해석: Biogen/ExpansionRX가 내부(proprietary) 데이터셋일 가능성이 높다. 이 경우 재현 불가 벤치마크에서의 성능 주장이 된다. ChEMBL-MT는 공개이므로 독립 검증 대상.

---

### 강점 / 한계

**강점**
- 코드 공개 — 즉시 적용 가능.
- 세 모달리티 통합으로 단일 모달리티 대비 화학 공간 커버리지 이론적 향상.
- 경량 설계(10만 분자 사전학습) — 컴퓨팅 비용 절감 주장.
- 지식 증류로 대형 교사 모델의 성능을 소형 배포 모델에 이식 가능.

**한계**
- 핵심 벤치마크(Biogen, ExpansionRX)가 내부 데이터일 가능성.
- KERMT 베이스라인의 공개 여부 및 구현 조건 불투명.
- MoleculeNet 등 공개 표준 벤치마크 결과 미기재(초록 기준).
- Finsler 기하학 융합의 실질적 기여 ablation 미확인.
- 10만 분자의 화학 다양성이 충분한지 미검증.

### 내 연구 적용 메모 (전략 노트)

- 전략 노트: 분자 특성 예측 파이프라인 구축 시 코드 공개 모델로 즉시 실험 가능. 단, 벤치마크 독립성 문제로 ChEMBL-MT 기준으로만 1차 평가 권장.
- 외부 맥락: ADME 예측은 약물 개발 초기 단계(hit-to-lead) 핵심으로, +7~10% 개선이 실제 스크리닝 히트율에 미치는 영향은 데이터 분포에 크게 의존한다.
- 질문: 분류(classification) vs 회귀(regression) 과제별 성능 분해 및 어떤 ADME 종말점(용해도, 막투과성, CYP 억제 등)에서 이득이 큰지 초록에 없음.

## 5. 확률론적 대조 사전학습 ADME 다중과제 예측

**논문**: "Probabilistic Contrastive Pretraining for Multi-task ADME Property Prediction"  
**링크**: https://arxiv.org/abs/2606.11508  
**상태**: arXiv preprint (미동료심사) · 2026-06-09  
**코드/데이터**: 미제공 — arXiv 페이지 및 초록에 저장소 링크 없음

---

### 무엇이 새로운가

**저자가 주장하는 새로운 점**:
- 재건(reconstruction) + 대조 판별(contrastive discriminative) + 화학 감독(chemical supervision) 세 학습 신호를 **단일 확률론적 잠재 변수 목적함수**로 통합.
- 분자 그래프 인코더로 잠재 표현을 생성하고 SMILES 디코더로 재건 — VAE 구조지만 확률론적 대조 학습이 추가.
- 다중 과제 GNN 헤드에서 과제별 MLP로 **음전이(negative transfer) 완화** 설계.

**실제로 새로운 점 구분**:
- 외부 맥락: 대조 학습(contrastive learning)은 MolCLR, GraphCL 등에서 분자에 적용됐다. 확률론적 잠재 변수와 대조 학습의 결합(VCCA 계열)도 존재한다.
- 해석: 이 논문의 기여는 세 목적함수의 통합 방식이지, 개별 구성 요소 자체는 새롭지 않다. 음전이 완화 설계가 ADME 다중과제에서 실제로 작동하는지가 핵심 검증 포인트다.
- 검토필요: GLACIER(논문 4)와 동일한 벤치마크 수치(Biogen +7.6%, ExpansionRX +9.9%, ChEMBL-MT +9.5%)를 보고. 두 논문이 같은 벤치마크를 같은 수치로 보고하는 것은 이례적이다. 공동 저자 관계 또는 동일 기관에서 같은 평가 셋을 사용했을 가능성 있음.

---

### 핵심 가정

1. **세 학습 신호의 결합이 상호 강화적**: 재건·대조·화학 감독이 서로 보완적이라고 가정하나, 목적함수 간 충돌(objective conflict)이 발생할 수 있다.
2. **잠재 변수가 화학 공간의 의미 있는 구조를 포착**: VAE 계열 모델에서 흔한 가정이나, 분자 활성 절벽(activity cliff)에서 잠재 공간이 불연속적으로 행동한다는 것이 알려져 있다(외부 맥락).
3. **과제별 MLP 헤드가 음전이를 충분히 차단**: 실제로 ADME 과제 간 상관관계(예: 용해도-투과성 트레이드오프)가 있어 완전한 분리가 최적이 아닐 수 있다.

1. 같은 기관/연구 그룹에서 공통 평가 플랫폼을 사용.
2. 두 논문이 서로의 비교 대상을 포함(즉, GLACIER가 이 논문을 베이스라인으로, 또는 반대).

초록만으로는 세 가지 가능성 중 어느 것인지 확인 불가. 논문 본문 접근 필요.

### 내 연구 적용 메모 (전략 노트)

- 전략 노트: GLACIER와 동시에 고려할 ADME 예측 방법이지만, 코드 미공개로 즉시 적용 불가. GLACIER 코드 실험 후 이 방법의 핵심 아이디어(확률론적 대조 사전학습)를 자체 구현하는 방향이 현실적.
- 질문: GLACIER와 이 논문의 저자 그룹 및 소속 기관 일치 여부 확인 필요. 같은 그룹의 병렬 논문이라면 앙상블 또는 직접 비교 실험이 가능할 것.
- 외부 맥락: ADME 예측에서 확률론적 표현의 실용 이점은 배치 스크리닝 시 상위 N개 후보의 불확실성 가중 순위(uncertainty-weighted ranking)에 있다.

## 교차 주제 메모

1. **파운데이션 모델의 벤치마크 독립성 문제**: W24 논문 다수가 내부(proprietary) 또는 한정된 벤치마크를 사용. 공개 표준 벤치마크(MoleculeNet, BindingDB 등) 병기가 없으면 주장의 일반성 평가 어렵다.

2. **코드 공개율**: 5편 중 2편(Span Detector, GLACIER)만 코드 공개. 즉시 적용 가능한 도구는 이 두 논문으로 좁혀진다.

3. **검출 한계 이하 데이터 처리(Span Detector)**는 임상 데이터 전반에 적용 가능한 일반 원칙으로, 논문 단독 가치 이상의 방법론적 시사점이 있다. LOD(Limit of Detection) 문제는 ADME 실험 데이터에도 동일하게 나타난다.

4. **GLACIER-ADME논문 동일 수치 이슈**: 두 논문이 같은 수치를 보고하는 것은 문헌 검색 시 같은 데이터 포인트를 두 번 세는 오류를 낳을 수 있다. 추후 저자 관계 확인 필요.
