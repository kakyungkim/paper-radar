# Source Grounding — 할루시네이션 방지 원칙 (Master)

paper-radar의 모든 에이전트(method/clinical/industry-analyst, claim-checker, digest-editor)가
**이 파일을 단일 진실 출처**로 참조한다. 각 에이전트는 이 규칙을 중복 기술하지 않고
"source-grounding 원칙을 따른다"로 링크만 건다.

---

## Part 1. 차단해야 할 것

- 초록·결과·방법 섹션·보충자료에 **없는** 수치·사실·인용을 원문의 내용처럼 쓰는 것.
- "이 분야에서는 일반적으로…"로 시작하는 외부 통념을 **근거 없이 단정**하는 것.
- 검색·웹 정보·기억으로 아는 다른 자료의 내용을 **현재 논문의 주장처럼** 쓰는 것.
- Figure caption만 보고 본문이 강조하지 않은 비교·해석을 **저자 주장**으로 쓰는 것.
- preprint를 peer-reviewed로, in silico 검증을 임상 검증으로 **수준을 높여** 표현하는 것.

## Part 2. 허용되는 것

- DOI로 접근 가능한 원문(초록·결과·방법·보충자료)에 명시적으로 나오는 수치·문장·그림·표.
- 본문이 직접 인용한 선행자료를 **그 선행자료 이름과 함께** 쓰는 것.
- 분석자(LLM)가 원문 근거 위에서 추론한 해석. 단 반드시 **Part 3 prefix**를 붙인다.

## Part 3. 외부 맥락·해석·질문의 명시 분리 (Prefix 6종)

원문 밖의 지식, 분석자 추론, 또는 follow-up을 써야 할 때는 아래 6가지 prefix 중 하나를 반드시 붙인다.

| Prefix | 의미 | 예시 |
|---|---|---|
| `해석:` | 원문 근거 위에서 분석자가 추론한 해석 | `해석: 이 모델의 AUC 0.91은 단일 기관 후향 코호트 기준이므로 외부 검증 전까지 과신 금지.` |
| `외부 맥락:` | 현재 논문 밖의 분야 일반 지식. 출처가 있으면 함께 | `외부 맥락: Foundation model pre-training 비용은 GPT-4 기준 ~$100M 수준 (OpenAI 2023 — 본문 미인용).` |
| `추정:` | 원문이 명시하지 않지만 문맥상 합리적 추론 | `추정: 학습 데이터에 외부 코호트가 포함됐을 가능성 있으나 본문에 명시 없음.` |
| `미제공:` | 원문이 해당 정보를 제공하지 않음 | `미제공: 코드 미공개. 재현 시도 어려움.` |
| `질문:` | 분석자가 다음 단계에서 확인해야 할 사항 | `질문: 이 모델이 한국 EHR 데이터에도 동일 성능인지 별도 검증 필요.` |
| `검토필요:` | 수치·사실이 의심스럽거나 출처 추적 실패 | `검토필요: 논문 본문과 초록의 AUC 수치가 다름 — 원문 재확인 요망.` |

모든 에이전트는 이 prefix 없이 원문 밖 지식을 **단정문으로 쓰지 않는다.**

## Part 4. 검증 수준 표기 (임상 논문 의무)

임상·바이오 논문은 claim이 어느 수준까지 검증됐는지를 반드시 표기한다.

| 수준 | 설명 | 표기 |
|---|---|---|
| in silico | 계산·시뮬레이션만 | `(in silico)` |
| 후향 (retrospective) | 기존 데이터로 사후 분석 | `(후향, N=…)` |
| 전향 (prospective) | 사전 설계 후 수집 | `(전향, N=…)` |
| 외부·다기관 | 다른 기관·코호트로 검증 | `(외부검증)` |
| RCT | 무작위 대조 시험 | `(RCT, N=…)` |

수준을 높여 표현(in silico → 임상, 후향 → 전향 등)하는 것은 **Part 1 위반**이다.

---

## Part 5. 출처 권위·신뢰 가중치 (non-paper 자료 포함)

논문이 아닌 자료(업계 보고서·뉴스·블로그·회사 IR)를 인용할 때:

| 등급 | 자료 유형 | 처리 |
|---|---|---|
| 1차 출처 | 원저 논문·FDA 서한·특허·임상시험 등록 | 그대로 인용 가능 |
| 2차 출처 | 학술 리뷰·시장조사 리포트 | `2차 출처:` 표기, 원저 추적 권장 |
| 의견·해설 | 뉴스·블로그·소셜 | `외부 맥락:` 또는 `해석:` prefix 필수 |

이해상충(COI): 저자·기관이 상업적 이익을 가진 자료는 `검토필요:` 플래그.
